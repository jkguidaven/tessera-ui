---
title: Design Tokens
description: CSS custom properties used across all Tessera UI components
---

All Tessera UI components use design tokens (CSS custom properties) for styling. Tokens use the `--ts-` prefix and cross the Shadow DOM boundary, making theming straightforward.

**Never use raw values in component CSS — always reference tokens.**

## Dark Mode

Dark mode is activated by adding `data-theme="dark"` to any ancestor. Token overrides are defined in `src/theme/tokens.css` and automatically adjust neutral colors and shadows for dark backgrounds.

```html
<html data-theme="dark">
  <!-- All components automatically use dark tokens -->
</html>
```
