## Principles

<div class="motion-principles">
  <div class="motion-principle">
    <div class="motion-principle-number">1</div>
    <div class="motion-principle-title">Communicate, don't decorate</div>
    <div class="motion-principle-desc">Every animation helps the user understand what happened — an element appeared, moved, or changed state.</div>
  </div>
  <div class="motion-principle">
    <div class="motion-principle-number">2</div>
    <div class="motion-principle-title">Fast by default</div>
    <div class="motion-principle-desc">Most interactions use 160ms or less. Users should never wait for an animation to finish.</div>
  </div>
  <div class="motion-principle">
    <div class="motion-principle-number">3</div>
    <div class="motion-principle-title">Consistent easing</div>
    <div class="motion-principle-desc">A shared set of easing curves creates a unified feel across all transitions.</div>
  </div>
  <div class="motion-principle">
    <div class="motion-principle-number">4</div>
    <div class="motion-principle-title">Respect preferences</div>
    <div class="motion-principle-desc">Animations are reduced or removed when prefers-reduced-motion is active.</div>
  </div>
</div>

### When to Use Each Duration

| Duration | Token | Use Cases |
|---|---|---|
| **80ms** | `--ts-transition-micro` | Checkbox tick, ripple start, instant-feel micro-feedback |
| **160ms** | `--ts-transition-fast` | Button hover, toggle switch, focus ring, color changes |
| **240ms** | `--ts-transition-normal` | Card hover elevation, component enter/exit |
| **360ms** | `--ts-transition-slow` | Accordion expand, panel slide, tab transitions |
| **500ms** | `--ts-transition-x-slow` | Page-level entrances, complex orchestrations |

## Reduced Motion

All animations in Tessera UI respect the `prefers-reduced-motion` media query. When enabled, transitions are either removed or replaced with instant opacity changes.

<div class="reduced-motion-card">
  <div class="reduced-motion-icon">A</div>
  <div class="reduced-motion-text">

    Some users experience vestibular disorders that make motion uncomfortable or disorienting. Always ensure animations degrade gracefully.
    <code class="reduced-motion-code">
      @media (prefers-reduced-motion: reduce) {"{"}<br/>
      &nbsp;&nbsp;*, *::before, *::after {"{"}<br/>
      &nbsp;&nbsp;&nbsp;&nbsp;animation-duration: 0.01ms !important;<br/>
      &nbsp;&nbsp;&nbsp;&nbsp;transition-duration: 0.01ms !important;<br/>
      &nbsp;&nbsp;{"}"}<br/>
      {"}"}
    </code>
  </div>
</div>
